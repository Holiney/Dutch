## Можливості

### 1) Irregular Verbs
- Практика всіх дієслів або вибраного діапазону.
- Введення форм **Imperfectum** та **Perfectum**.
- Статуси перевірки: `correct`, `warning`, `incorrect`.
- Підсумок раунду з розбором помилок.

### 2) Words Quiz
- Перевірка перекладу базових слів.
- Формат карток з перевіркою відповіді.
- Підсумок раунду з помилками.

### 3) НОВЕ: Words Study Flashcards
- Окремий режим вивчення слів без тесту.
- Гортання карток, показ/приховування перекладу.
- Позначення слів як вивчених з індикатором прогресу.

### 4) Прогрес навчання
- Збереження прогресу в `localStorage` для режиму дієслів.
- Метрики: кількість раундів, кількість опрацьованих слів, точність, найкращий раунд.

## Локальний запуск

### Вимоги
- Node.js 20+
- npm

### Кроки
1. Встановити залежності:
   ```bash
   npm install
   ```
2. Запустити dev-сервер:
   ```bash
   npm run dev
   ```
3. Відкрити застосунок у браузері:
   - `http://localhost:3000`

## Команди

```bash
npm run dev      # запуск у режимі розробки
npm run build    # production-збірка
npm run preview  # превʼю зібраного застосунку
npm run lint     # перевірка TypeScript
```
