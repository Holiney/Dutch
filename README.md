## Можливості

### 1) Irregular Verbs
- Практика всіх дієслів або вибраного діапазону.
- Вибір діапазону **від слова до слова** (а не за номером).
- Введення форм **Imperfectum** та **Perfectum**.
- Статуси перевірки: `correct`, `warning`, `incorrect`.

### 2) Words: 3 режими
1. **Study Cards**: нідерландське слово + переклад + транскрипція одразу.
2. **Quiz UKR → NL**: бачиш українське слово, пишеш нідерландською.
3. **Quiz NL → UKR**: бачиш нідерландське слово, пишеш українською.

### 3) Прогрес навчання
- Збереження прогресу в `localStorage` для режиму дієслів.
- Метрики: кількість раундів, кількість опрацьованих слів, точність, найкращий раунд.

## Локальний запуск

### Вимоги
- Node.js 20+
- npm

### Кроки
1. Встановити залежності:
   ```bash
   npm install
   ```
2. Запустити dev-сервер:
   ```bash
   npm run dev
   ```
3. Відкрити застосунок у браузері:
   - `http://localhost:3000`

## Команди

```bash
npm run dev      # запуск у режимі розробки
npm run build    # production-збірка
```
